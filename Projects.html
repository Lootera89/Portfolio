Rgames - Fabwelt Studio

Unreal Engine AI and Physics AA Project

I joined Fabwelt Studios in April 2023 as their first Unreal Engine developer to work on a racing game project. Fabwelt, known for its blockchain FPS games like Arsenal, aimed to create an MVP AA-level racing game to secure funding for further development.

I worked on this project for six months, handling AI, car physics, UI design, and optimization. The game features dynamic car AI, advanced motion blur, and realistic physics, all designed to run smoothly on low-end PCs while offering an immersive experience.

  • Reinforcement Learning AI with PPO for opponent cars
  • Advanced minimap, speedometer, and UI/UX
  • Custom motion blur and other visual effects
  • Realistic car physics and controls
  • World design, level streaming, and Niagara effects

Demono: Home Alone - Lootera89

Unreal Engine Atmospheric Design Solo Project

Demono: Home Alone is a short, story driven horror game I developed solo, starting in March 2024. The story was written by two friends and voiced by a professional artist, I handled all aspects of the game's design and development.

This game focuses on creating an immersive atmosphere using custom PSX-style dithering effects and layered sound design. Despite its detailed visuals and interactive elements, the game is highly optimized with a final size of just 400 MB.

  • Complex dithering post process material for PSX aesthetics
  • Many player to world interactive objects
  • Scalable and dynamic quest system
  • Level streaming for optimization
  • Layered sound design for an immersive atmosphere

Fistful Of Gore - Lootera89

Unreal Engine Cross Platform Solo Project

Fistful of Gore is a fast-paced beat 'em up game for Android and Windows. I started developing it in April 2023. Players grow stronger as they fight tougher enemies across five different maps, each with a boss at the end.

The game includes custom body dismemberment, 2D blood particles, health bars, and a global leaderboard. Right now, I'm working on character art, background art with an artist, and improving enemy types, bosses, and mobile controls.

  • Advanced AI using Learning Agent and PPO algorithm
  • Mobile and PC cross-platform optimized controls
  • Custom body dismemberment system with 2D blood effects
  • Global leaderboard, health bars and damage meters
  • Unique enemies variation

Hidden Beyond - Yellow Whale Labs

Unreal Engine Multiplayer AA Project

Hidden Beyond is a multiplayer blockchain game currently under development by Yellow Whale Labs. I joined Yellow Whale Labs in December 2023 as a multiplayer and backend developer. I worked there for 4 months. The game is still in development.

I worked on multiplayer systems like setting up P2P servers and using EOS, Playfab, and AWS. I made character creation, main menu UI, and a login-signup system. I also migrated Metahuman models to game characters and focused on optimizing the game.

  • Set up P2P multiplayer servers
  • Worked with EOS, Playfab, and AWS
  • Created login and sign up system
  • Migrated Metahuman models to characters
  • Optimized performance and backend systems
